Setup Flutter

Introduction

We are using flutter which allows having complete automation for build flow. In case if you don't know Gulp then it's easy to use it. Gulp is a toolkit for automating painful or time-consuming tasks in the development workflow, so you can stop messing around while building any project. You can read it more about it here. Please follow below steps to install and setup all prerequisites:

Prerequisites

Please follow below steps to install and setup all prerequisites:

  • Flutter

    Make sure to have the Flutter SDK installed & running in your computer. If you already have installed flutter on your computer, you can skip this step.

  • Note

    Make sure to have the Update path in your computer, For the more information Environment Path Update & in your computer. If you already have Update Environment Path in your computer, you can skip this step.

Installation

To setup the admin theme, follow below-mentioned steps:

  • Install Prerequisites

    Make sure to have all above prerequisites installed & running on your computer.

    1. Make sure to have VS Code or Android Studio install in your computer, If you already have installed on your computer, you can skip this step.

    2. After successfully installed VS Code or Android Studio in your computer then you need to open Vs Code or Android Studio from your computer and install dart and flutter extension in your VS Code or Android Studio.

    3. After successfully install dart and flutter extension. open project in VS Code or Android Studio from file menu.

    4. After open project in ( VS Code see the menu bar and click Terminal -> New Terminal ) -OR- ( Android Studio see the footer and click Terminal) after open terminal run flutter pub get command in terminal. now ready for project run.

    5. For VSCode see the menu bar Run option and click on Run -> start debugging now project run. -or- For Android Studio see the menu bar Run option and click on Run -> Run 'main.dart' now project run.
